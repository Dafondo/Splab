## Usage
Remember, Python2 only.  
1. Clone this repository
2. Install requirements  
      `pip install -r server/requirements.txt` 
3. Launch simple server (on server)  
      `./launch-simple-server.sh`
4.  Launch clients  
      `python2 -m SimpleHTTPServer 25125`
5. Play Game
